Simple and unified UI for all journals

Displays papers across from various journals in simple and unform layout. Works on a web browser. No standalone software needed.

Fast, smooth, and automatic

Blazingly fast browsing by keyboard shortcuts. All full texts are automatically saved when you visit online journal articles (you can disable this if you want).

EPUB export

Papers can be exported as portable EPUB files. You can even read papers on your iPhone!

Library import & export

You can import an existing library.
